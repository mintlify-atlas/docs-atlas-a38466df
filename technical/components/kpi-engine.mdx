## Descripción

El motor de KPIs (`kpi_engine.py`) es el componente central para el cálculo de todas las métricas comerciales de Visor KPI. Procesa DataFrames ya filtrados y retorna estructuras listas para visualización.

**Responsabilidades principales:**
- Cálculo de KPIs gerenciales (ventas, margen, cobertura)
- Métricas por vendedor con rankings y semáforos
- Análisis de productos y clientes con Pareto
- Evolución temporal y comparaciones interperiodo
- Generación de insights ejecutivos automáticos

## KPIs Gerenciales

### kpi_ventas_periodo()

Calcula métricas agregadas de ventas para un período específico.

```python
def kpi_ventas_periodo(
    df: pd.DataFrame,
    df_anterior: pd.DataFrame | None = None,
    objetivos_df: pd.DataFrame | None = None,
    fecha_desde: date | None = None,
    fecha_hasta: date | None = None,
) -> dict
```

<ParamField path="df" type="pd.DataFrame" required>
  DataFrame de ventas filtrado para el período actual
</ParamField>

<ParamField path="df_anterior" type="pd.DataFrame | None" default="None">
  DataFrame del período anterior para comparaciones
</ParamField>

<ParamField path="objetivos_df" type="pd.DataFrame | None" default="None">
  DataFrame de objetivos con columnas `periodo` y `objetivo`
</ParamField>

<ParamField path="fecha_desde" type="date | None" default="None">
  Fecha de inicio del período para filtrar objetivos
</ParamField>

<ParamField path="fecha_hasta" type="date | None" default="None">
  Fecha final del período para filtrar objetivos
</ParamField>

<ResponseField name="return" type="dict">
  Diccionario con las siguientes claves:
  
  <Expandable title="Estructura del diccionario">
    - `importe_total` (float): Suma de ventas netas del período
    - `importe_vs_anterior` (float | None): Variación relativa vs período anterior (0.15 = +15%)
    - `objetivo_total` (float): Suma de objetivos del período
    - `cumplimiento_pct` (float | None): Porcentaje de cumplimiento (1.0 = 100%)
    - `transacciones_count` (int): Número de transacciones
    - `ticket_promedio` (float): Venta promedio por cliente único
    - `clientes_unicos` (int): Cantidad de clientes distintos
  </Expandable>
</ResponseField>

**Ejemplo de uso:**

```python
from src.kpi_engine import kpi_ventas_periodo
from datetime import date

kpis = kpi_ventas_periodo(
    df=ventas_marzo,
    df_anterior=ventas_febrero,
    objetivos_df=objetivos,
    fecha_desde=date(2026, 3, 1),
    fecha_hasta=date(2026, 3, 31)
)

print(f"Ventas: ${kpis['importe_total']:,.0f}")
print(f"Cumplimiento: {kpis['cumplimiento_pct']*100:.1f}%")
```

## KPIs por Vendedor

### kpi_por_vendedor()

Genera DataFrame completo con todos los KPIs por vendedor incluyendo rankings y semáforos.

```python
def kpi_por_vendedor(
    df: pd.DataFrame,
    objetivos_df: pd.DataFrame,
    df_anterior: pd.DataFrame | None = None,
    fecha_desde: date | None = None,
    fecha_hasta: date | None = None,
) -> pd.DataFrame
```

<ParamField path="df" type="pd.DataFrame" required>
  DataFrame de ventas con columnas: `id_vendedor`, `importe_neto`, `id_cliente`, `nombre_vendedor`, `zona_vendedor`, `perfil`
</ParamField>

<ParamField path="objetivos_df" type="pd.DataFrame" required>
  Objetivos por vendedor y período
</ParamField>

<ParamField path="df_anterior" type="pd.DataFrame | None" default="None">
  Ventas del período anterior para deltas
</ParamField>

<ParamField path="fecha_desde" type="date | None" default="None">
  Fecha inicio para filtrar objetivos
</ParamField>

<ParamField path="fecha_hasta" type="date | None" default="None">
  Fecha final para filtrar objetivos
</ParamField>

<ResponseField name="return" type="pd.DataFrame">
  DataFrame con una fila por vendedor y las siguientes columnas:
  
  <Expandable title="Columnas del DataFrame">
    - `id_vendedor` (str)
    - `nombre_vendedor` (str)
    - `zona_vendedor` (str)
    - `perfil` (str)
    - `importe_total` (float)
    - `transacciones` (int)
    - `clientes_únicos` (int)
    - `objetivo` (float)
    - `cumplimiento_pct` (float | None)
    - `ticket_promedio` (float)
    - `importe_vs_mes_anterior` (float | None)
    - `ranking_actual` (int)
    - `ranking_anterior` (int)
    - `delta_ranking` (int): Diferencia de ranking (positivo = mejoró)
    - `color_semaforo` (str): Color hex para visualización
    - `label_semaforo` (str): `"success"`, `"warning"`, o `"danger"`
  </Expandable>
</ResponseField>

**Lógica de semáforo:**
- 🟢 Verde (`success`): cumplimiento ≥ 100%
- 🟡 Naranja (`warning`): cumplimiento ≥ 80%
- 🔴 Rojo (`danger`): cumplimiento < 80%

**Ejemplo:**

```python
vendedores = kpi_por_vendedor(
    df=ventas_filtradas,
    objetivos_df=objetivos,
    df_anterior=ventas_mes_anterior,
    fecha_desde=periodo_desde,
    fecha_hasta=periodo_hasta
)

# Top 5 vendedores
top5 = vendedores.head(5)
for _, v in top5.iterrows():
    print(f"{v['nombre_vendedor']}: ${v['importe_total']:,.0f} ({v['cumplimiento_pct']*100:.1f}%)")
```

### kpi_clientes()

Genera análisis completo por cliente con Pareto y flags de riesgo.

```python
def kpi_clientes(
    df: pd.DataFrame,
    clientes_df: pd.DataFrame,
    df_anterior: pd.DataFrame | None = None,
    fecha_hasta: date | None = None,
) -> pd.DataFrame
```

<ParamField path="df" type="pd.DataFrame" required>
  Ventas del período con `id_cliente`, `importe_neto`, `fecha`
</ParamField>

<ParamField path="clientes_df" type="pd.DataFrame" required>
  Maestro de clientes
</ParamField>

<ParamField path="df_anterior" type="pd.DataFrame | None" default="None">
  Ventas período anterior
</ParamField>

<ParamField path="fecha_hasta" type="date | None" default="None">
  Fecha de referencia para calcular días desde última compra
</ParamField>

<ResponseField name="return" type="pd.DataFrame">
  <Expandable title="Columnas">
    - `id_cliente` (str)
    - `razon_social` (str)
    - `canal` (str)
    - `zona` (str)
    - `nombre_vendedor` (str)
    - `importe_total` (float)
    - `frecuencia_compra` (int): Cantidad de compras
    - `ticket_promedio` (float)
    - `ultima_compra` (Timestamp)
    - `primera_compra` (Timestamp)
    - `dias_desde_ultima_compra` (int)
    - `categoria_pareto` (str): "A", "B", o "C"
    - `participacion_pct` (float)
    - `acumulado_pct` (float)
    - `vs_periodo_anterior` (float | None)
    - `flag_en_riesgo` (bool): True si categoría A/B con >30 días sin compra
  </Expandable>
</ResponseField>

**Flag de riesgo:**

Se marca `flag_en_riesgo=True` cuando:
- Cliente es categoría **A** o **B** en Pareto
- Y tiene **más de 30 días** sin comprar

## Notas de Implementación

<Note>
  Todas las funciones esperan DataFrames **ya filtrados** por fecha, vendedor, zona, etc. 
  El filtrado se realiza en `data_loader.py` antes de invocar el motor de KPIs.
</Note>

<Warning>
  Los valores de cumplimiento y variaciones pueden ser `None` cuando:
  - No hay objetivos definidos
  - No hay datos del período anterior
  - El denominador es cero (división por cero)
  
  Manejar estos casos en la capa de visualización.
</Warning>
