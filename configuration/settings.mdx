## Overview

All global configuration for Visor KPI Comercial is centralized in `config.py`. This file defines colors, business constants, formatting functions, data paths, and Plotly theme settings.

## Formatting Functions

The configuration includes several utility functions for consistent number formatting:

### Currency Formatting

```python
fmt_moneda(1284500)
# Returns: "$1.284.500"
```

Formats numbers as Argentine peso currency with thousands separators (dots) and no decimals for values >= 1000.

### Percentage Formatting

```python
fmt_pct(0.842, decimales=1)
# Returns: "84.2%"
```

Formats decimal values as percentages with configurable decimal places.

### Number Formatting

```python
fmt_numero(1284500)
# Returns: "1.284.500"
```

Formats numbers with thousands separators using dots.

### Delta Formatting

```python
fmt_delta(0.082)
# Returns: "▲ +8.2%"

fmt_delta(-0.031)
# Returns: "▼ -3.1%"
```

Formats variance with directional arrows. Values between -0.05% and +0.05% display as "— 0.0%".

### HTML Delta Formatting

```python
fmt_delta_html(0.082)
# Returns: '<span style="color:#00C49F">▲ +8.2%</span>'
```

Similar to `fmt_delta` but includes inline HTML color styling:
- Positive: green (#00C49F)
- Negative: red (#E84855)
- Neutral: gray (#8899AA)

## Plotly Theme

Custom Plotly theme settings for consistent chart styling:

```python
PLOTLY_THEME = {
    "paper_bgcolor": COLORS["bg_card"],
    "plot_bgcolor":  COLORS["bg_card"],
    "font": {
        "color": COLORS["text_primary"],
        "family": "Inter, sans-serif",
        "size": 12
    },
    "margin": dict(l=20, r=20, t=40, b=20)
}
```

Apply this theme to Plotly figures for consistent appearance across all charts.

## Date Period Utilities

The configuration provides helper functions for working with date periods:

### Available Periods

```python
get_periodos()
# Returns dict with predefined periods:
# {
#     "Este mes": (first_day_of_month, today),
#     "Mes anterior": (first_day_last_month, last_day_last_month),
#     "Último trimestre": (3_months_ago, today),
#     "Este año": (jan_1, today),
#     "Año anterior": (jan_1_last_year, dec_31_last_year),
#     "Personalizado": None
# }
```

### Previous Period Calculation

```python
get_periodo_anterior(fecha_desde, fecha_hasta)
# Returns tuple with equivalent previous period dates
```

Calculates a comparison period of the same duration immediately preceding the given date range.

## Example Configuration

Here's how to use the configuration in your modules:

```python
from config import COLORS, THRESHOLDS, fmt_moneda, fmt_pct

# Use colors
header_color = COLORS["primary"]
success_color = COLORS["success"]

# Format values
revenue_display = fmt_moneda(1250000)  # "$1.250.000"
achievement = fmt_pct(0.95)            # "95.0%"

# Check thresholds
if performance >= THRESHOLDS["cumplimiento_ok"]:
    status_color = COLORS["success"]
```
