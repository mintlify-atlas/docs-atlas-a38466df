## Overview

The **Management View** (Vista Gerencial) provides commercial directors and sales managers with comprehensive tools to monitor, analyze, and optimize their entire sales team's performance. This view consolidates individual seller metrics, territorial analysis, and trend identification in a single management dashboard.

<Info>
  **Who uses this view**: Commercial managers, regional directors, and team leaders responsible for sales team performance and territory management.
</Info>

## Team Performance Analysis

### Monthly Evolution Chart

The **Team Monthly Evolution** chart displays:

- Historical sales trend (up to 13 months)
- Actual performance vs monthly targets
- Cumulative view of team progress
- Seasonal pattern identification

**Use cases**:
- Identify growth/decline trends
- Spot seasonal patterns
- Forecast future performance
- Set realistic targets

### Seller Performance Comparison

The **Performance by Seller** bar chart shows:

- Each seller's quota attainment percentage
- Color-coded bars:
  - 🟢 **Green** (100%+): Meeting or exceeding target
  - 🟡 **Orange** (80-99%): Close to target
  - 🔴 **Red** (&lt;80%): Below expectations
- Horizontal reference line at 100%
- Sortable by performance

<Note>
  This chart makes it instantly clear which sellers need coaching or support versus those who could mentor others.
</Note>

## Complete Seller Ranking

The **full seller ranking table** provides detailed performance data:

### Columns Displayed

| Column | Description |
|--------|-------------|
| **Rank** | Current position (1 = highest sales) |
| **Seller Name** | Full name and zone assignment |
| **Total Sales** | Revenue generated in period |
| **Quota Attainment** | % of assigned target achieved |
| **Transactions** | Number of sales completed |
| **Unique Clients** | Distinct customers served |
| **Avg Ticket** | Revenue per unique client |
| **vs Previous Period** | % change from prior period |
| **Rank Movement** | ↑/↓ position change |
| **Profile** | Seller classification (star/stable/development/problematic) |

### Traffic Light System

Each row is highlighted with a performance indicator:

- 🟢 **Green**: 100%+ attainment (target exceeded)
- 🟡 **Orange**: 80-99% attainment (approaching target)
- 🔴 **Red**: &lt;80% attainment (needs attention)

<Tip>
  Click any column header to sort the table. Use this to quickly identify best performers or those needing support.
</Tip>

**Insights you can gain**:
- Which days are most productive for your team

## Management Workflows

### Weekly Team Review

**Every Monday morning**:

1. **Check Team KPIs**
   - Are we on track for monthly target?
   - What's the average ticket trend?

2. **Review Seller Rankings**
   - Who moved up/down in rankings?
   - Who needs coaching this week?

3. **Analyze Activity Patterns**
   - Are there concerning gaps in the heatmap?
   - Which days need activity boosts?

4. **Territory Performance**
   - Which zones are underperforming?
   - Where should we focus resources?

### Monthly Performance Reviews

**End-of-month routine**:

1. Compare actual vs target for each seller
2. Identify top performers for recognition
3. Schedule coaching sessions for underperformers
4. Analyze trend over last 3-6 months
5. Adjust next month's targets if needed

### Coaching Trigger Points

Automatic coaching triggers based on KPIs:

<Warning>
  **Critical (immediate action)**:
  - Seller below 65% attainment
  - Zero activity for 7+ business days
  - Ranking drop of 3+ positions
</Warning>

<Info>
  **Coaching opportunity (schedule session)**:
  - 2 consecutive months in orange zone
  - Declining trend over 3 months
  - High transaction count but low avg ticket
</Info>

## KPI Calculations

All metrics follow these calculation rules:

### Team Sales Total
```
Sum of importe_neto for all transactions
where id_vendedor in selected_sellers
and fecha between fecha_desde and fecha_hasta
```

### Quota Attainment
```
(Actual Sales / Assigned Target) × 100
where target = sum of monthly objectives for seller
```

### Average Ticket
```
Total Sales / Unique Clients Served
```

### Rank Calculation
```
DESC order by importe_total
Current rank - Previous rank = Delta
```

### Traffic Light Logic
```
if attainment >= 100%: Green
else if attainment >= 80%: Orange
else: Red
```
